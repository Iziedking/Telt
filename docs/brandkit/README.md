# Telt brand kit

The start of the brand kit. More assets to come.

## Logo lockup

The circular t-in-ring emblem (the ring and thick `t` in the ink, the dot in Signal red),
beside the **telt** wordmark in Fredoka (the brand display face): `tel` in ink, the final
`t` in Signal red.

| File | Use |
| --- | --- |
| `telt-lockup.png` | transparent background, ink wordmark (place on light surfaces) |
| `telt-lockup-light.png` | on the cream brand surface (`#F4F1EA`) |
| `telt-lockup-dark.png` | on the ink brand surface (`#14181F`) |
| `telt-lockup.svg` | vector source (wordmark needs the Fredoka font installed) |

## Colors

- Ink `#14181F` · Cream `#F4F1EA` · Signal red `#E8352B`
- Pastels: felt `#A8E0C2` · periwinkle `#C7C9F2` · sky `#BFE0F2` · sand `#F1E7CE`

## Type

- Display: **Fredoka** (rounded, chunky)
- Body: Hanken Grotesk · Mono: Space Mono

## Regenerating / new assets

The PNGs were rendered from the badge shapes plus Fredoka glyph outlines. The live lockup
also renders at `/brand` in the frontend (Fredoka loaded), which is handy for screenshots.
